## 🚀 Quick Start
1. Extract the archive
2. Install the correct version for your Minecraft (Forge or Fabric)
3. Place the .jar file into your mods folder
4. Launch Minecraft
5. Spawn Jenny using the command: /jenny spawn

> ℹ️ Complete Jenny Mod collection for all Minecraft versions.
